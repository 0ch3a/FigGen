# [Installation]
```
$ pkg update upgrade
$ pkg install python2
$ git clone https://github.com/0ch3a/FigGen.git
$ cd FigGen
```
# [Setup]
```
$ python2 install.py
```
# [Running]
```
$ python2 FigGen.py
```
